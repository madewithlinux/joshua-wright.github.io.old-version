My Favorite Projects:

WebGL Fractals

Animation showing the intersection of the Mandelbrot and Julia sets. Rendering is done using WebGL controlled by JavaScript.

RPN Calculator

A basic RPN calculator implemented in JavaScript. Automatically generates an expression tree and infix form as you type an RPN expression.

Image Stenography

A console application which hides a binary or text payload inside the least-significant bits of a .png image. Uses zlib compression to shrink the payload.

xkcd App

Android app to view the excellent web comic xkcd. Uses xkcd's JSON API, with caching to avoid re-downloading comics.

JavaScript Examples

Some of the early JavaScript/jQuery examples I have written. Currently includes a number factoring example, an implementation of the Sieve of Eratosthenes prime number algorithm, and a RPN Calculator with real-time visible stack
