Iterated Affine Transform Fractals

Iterated affine transform fractals rendered in JavaScript. Uses a constant-space non-recursive algorithm, and dynamically generates a function for each fractal with folded constants.

RPN Calculator

A basic RPN calculator implemented in JavaScript. Automatically generates an expression tree and infix form as you type an RPN expression.

Image Stenography

A console application which hides a binary or text payload inside the least-significant bits of a .png image. Uses zlib compression to shrink the payload.
